# Synced Walkthrough Decks

This bundle contains:

- `a3_walkthrough_presenter.tex` — main audience slideshow
- `a3_walkthrough_speaker.tex` — private synced notes deck

Both decks are synchronized slide-for-slide (32 frames including title).

## Compile

From `agential_demo/`:

```bash
pdflatex a3_walkthrough_presenter.tex
pdflatex a3_walkthrough_speaker.tex
```

## Present workflow

1. Put `a3_walkthrough_presenter.pdf` on the projector/shared screen.
2. Keep `a3_walkthrough_speaker.pdf` on your private screen.
3. Advance both together by frame number.
4. Budget ~2 minutes per slide for a ~60-minute deep dive.
